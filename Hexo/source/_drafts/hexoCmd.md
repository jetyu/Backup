常用的命令
hexo clean;清除public下内容
hexo g;生成文章
hexo s;启动服务器
hexo server --drafts  连草稿内容启动
hexo clean;hexo g;hexo s;启动服务器
hexo clean;hexo g;hexo d;发布文章
